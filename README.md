# CartPole Q-Learning

## Description
This project implements a Q-learning agent for the CartPole reinforcement learning environment.

The agent learns a control policy using discretized states and epsilon-greedy exploration, then is evaluated after training.

## Technologies
- Python
- NumPy
- Gym

## Features
- Q-learning algorithm
- State discretization
- Epsilon-greedy action selection
- Training over multiple episodes
- Policy evaluation after training
